# 📚 JavaScript for Everyone — Practice Code

> Practice code based on the GitHub repository:
> **[Asabeneh/JavaScript-for-Everyone](https://github.com/Asabeneh/JavaScript-for-Everyone)**
>
> A step-by-step guide to learn JavaScript from scratch to advanced concepts.

---

## 🚀 How to Run

Make sure you have **Node.js** installed.
To check: open a terminal and type:

```bash
node -v
```

Then navigate into any day folder and run:

```bash
node day1.js
```

Or from the root JavaScript folder:

```bash
node Day1/day1.js
```

## 📅 Day-by-Day Topics

| Day    | Topics Covered |
|--------|----------------|
| Day 01 | `console.log`, Comments, Arithmetic, Variables (`let`, `const`) |
| Day 02 | Primitive vs Non-Primitive types, Numbers, `Math` object, Strings |
| Day 03 | `.includes()`, `.replace()`, `.match()`, `typeof`, Type Casting |
| Day 04 | Arithmetic & Logical operators, `if/else`, `switch`, Ternary |
| Day 05 | `for`, `while`, `do...while` loops |
| Day 06 | Arrays, `.push()`, `.pop()`, `.slice()`, `.sort()`, `.reverse()` |
| Day 07 | Function declaration, Arrow functions, IIFE, Rest params |
| Day 08 | Object literals, `Object.keys/values/entries/assign`, Date |
| Day 09 | `forEach`, `map`, `filter`, `reduce`, `every`, `some`, `find` |
| Day 10 | Destructuring, Spread `...`, `Set`, `Map`, `class`, Inheritance |
| Day 11 | `JSON.parse()`, `JSON.stringify()`, reviver, filter array |
| Day 12 | getter, setter, static methods, method overriding, `sort()` with comparator |
| Day 13 | RegEx: `.test()`, `.match()`, `.replace()`, `[]`, `\d`, `^$`, `{}` |
| Day 14 | Closures, Promises, `async/await`, `try/catch/finally` |
| Day 15 | `localStorage`, `sessionStorage`, `document.cookie` (with Node.js simulation) |
